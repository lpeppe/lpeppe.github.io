Projects

MeteoWhere

MeteoWhere is a hybrid application developed using Ionic 3. It allows users to easily find places to do various activities based on the forecast. Thanks to MeteoWhere, for example, it's easy to find ski stations to snowboard and the app will suggest only the ski slopes where the snow is high enough! The app has been developed for Spinvector CEO Giovanni Caturano.

The project was developed with Maria Elena Cammarano.

SoMusic!

SoMusic! is a plugin built for the Social Networking Service Oxwall. It allows users to create and share musical scores. SoMusic! has a built-in music editor and a graphic visualization tool which helps users finding errors made during composition, using the 4-part music theory.

I developed the score editor from scratch using vanilla Javascript and then I used it to build an Oxwall plugin written in PHP.

DiscoverItaly

DiscoverItaly is a hybrid app developed using Ionic 3 which allows users to find places of interest, restaurants, typical local products and festivals nearby!

The app was developed as a web scraping project for an exam and I worked on both the front-end and the back-end. The back-end is built with NodeJS, uses MongoDB for persistence and was deployed on AWS. Data was scraped from many websites, among which TripAdvisor. To do that, I developed a bot using PhantomJS headless browser, Selenium Webdriver and NodeJS.

MPI Matrix Multiplication

This simple project was created for a parallel programming exam. The program takes as input two matrix A (m x n) and B (n x l) and multiplies them spreading the computation across multiple processors. Any number of processors can be used for the computation.

The project was developed using MPI and deployed on a cluster hosted on AWS.

Auction Mechanism

This simple project was created for a p2p networks exam. It implements a Second-Price Auction System using a peer to peer network.

The project was developed in Java using the library TomP2P together with Maria Elena Cammarano.
