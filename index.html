SoMusic!

SoMusic! is a plugin built for the Social Networking Service Oxwall. It allows users to create and share musical scores. SoMusic! has a built-in music editor and a graphic visualization tool which helps users finding errors made during composition, using the 4-part music theory.

MeteoWhere

MeteoWhere is a hybrid application developed using Ionic 3. It allows users to easily find places to do various activities based on the forecast. Thanks to MeteoWhere, for example, it's easy to find ski stations to snowboard and the app will suggest only the ski slopes where the snow is high enough! The app has been developed for Spinvector CEO Giovanni Caturano.
